  <name>Task 1: Segmentação RMR (8 segmentos) e faixas de GAP (5 faixas)</name>
  <files>src/segmentation.py, tests/test_segmentation.py</files>
  <behavior>
    8 segmentos RMR baseados em score_R, score_M e score_Ritmo (escala 1–5):
    - Campeões: R ≥ 4 e M ≥ 4 e Ritmo ≥ 4
    - Leais: R ≥ 3 e M ≥ 3 e Ritmo ≥ 3 (não Campeões)
    - Potenciais Leais: R ≥ 3 e M ≥ 2 (não nas anteriores)
    - Novos Clientes: R == 5 e (M <= 2 ou Ritmo <= 2)
    - Em Risco: R <= 2 e M ≥ 3
    - Hibernando: R <= 2 e M <= 2
    - Não Pode Perder: R <= 2 e M == 5
    - Precisam Atenção: todos os demais elegíveis
    (Usar checagem em ordem de prioridade: Campeões → Não Pode Perder → Leais → Novos Clientes → Potenciais Leais → Em Risco → Hibernando → Precisam Atenção)

    5 faixas de GAP (limiares padrão configuráveis):
    - Crítico: GAP < -30
    - Atrasado: -30 <= GAP < -8
    - Hora de Comprar: -8 <= GAP <= 0
    - Em Breve: 0 < GAP <= 7
    - Folgado: GAP > 7
    - Inelegível: GAP é NaN (cliente de 1 compra)

    Testes:
    - Test: score_R=5, score_M=5, score_Ritmo=5 → segmento "Campeões"
    - Test: score_R=1, score_M=5 → segmento "Não Pode Perder"
    - Test: score_R=1, score_M=2 → segmento "Hibernando"
    - Test: GAP=-40 → faixa "Crítico"
    - Test: GAP=-15 → faixa "Atrasado"
    - Test: GAP=0 → faixa "Hora de Comprar"
    - Test: GAP=5 → faixa "Em Breve"
    - Test: GAP=20 → faixa "Folgado"
    - Test: GAP=NaN → faixa "Inelegível"
    - Test: cliente com scores NaN (Ritmo NaN) → segmento "Inelegível"
  </behavior>
  <action>
    Criar src/segmentation.py com:

    `DEFAULT_GAP_THRESHOLDS` — dict com limiares padrão:
    ```python
    DEFAULT_GAP_THRESHOLDS = {
        "critico_max": -30,      # GAP < -30 → Crítico
        "atrasado_max": -8,      # -30 <= GAP < -8 → Atrasado
        "hora_max": 0,           # -8 <= GAP <= 0 → Hora de Comprar
        "em_breve_max": 7,       # 0 < GAP <= 7 → Em Breve
        # GAP > 7 → Folgado
    }
    ```

    `classify_rmr_segment(row) -> str`:
    - Recebe uma linha do DataFrame RMR (com score_R, score_M, score_Ritmo)
    - Se qualquer score é NaN: retorna "Inelegível"
    - Aplica as 8 regras em ordem de prioridade (usar if/elif encadeado)
    - Retorna nome do segmento como string

    `classify_gap_status(gap_value, thresholds=None) -> str`:
    - Recebe valor float de GAP (pode ser NaN) e dict de thresholds
    - Se thresholds=None: usa DEFAULT_GAP_THRESHOLDS
    - Se gap_value é NaN: retorna "Inelegível"
    - Classifica nas 5 faixas e retorna string

    `apply_segmentation(rmr_df, gap_thresholds=None) -> pd.DataFrame`:
    - Recebe rmr_df (output de compute_rmr com scores)
    - Adiciona coluna "Segmento_RMR" usando classify_rmr_segment em cada linha
    - Adiciona coluna "Faixa_GAP" usando classify_gap_status em cada linha
    - Retorna DataFrame com as duas novas colunas
