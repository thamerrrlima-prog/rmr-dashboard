          - `elegíveis = result_df[result_df["Segmento_RMR"] != "Inelegível"].copy()`
          - Se ID estiver no índice: `elegíveis = elegíveis.reset_index().rename(columns={"index": "ID"})` (ajustar nome conforme real)
          - `elegíveis["Ticket Médio"] = elegíveis["Monetario"]`  — renomear para exibição
          - Criar três controles de filtro lado a lado com st.columns(3):
            * col1: `sel_playg = st.multiselect("PlayG", sorted(elegíveis["PlayG"].dropna().unique()), key="lista_playg")`
            * col2: `sel_faixa = st.multiselect("Classificação GAP", ["Crítico", "Atrasado", "Hora de Comprar", "Em Breve", "Folgado"], key="lista_faixa")`
            * col3: `busca = st.text_input("Buscar nome ou ID", key="lista_busca", placeholder="Digite nome ou ID...")`

       b) Aplicar filtros com lógica AND:
          ```python
          df_lista = elegíveis.copy()
          if sel_playg:
              df_lista = df_lista[df_lista["PlayG"].isin(sel_playg)]
          if sel_faixa:
              df_lista = df_lista[df_lista["Faixa_GAP"].isin(sel_faixa)]
          if busca.strip():
              termo = busca.strip().lower()
              mask_nome = df_lista["Nome"].str.lower().str.contains(termo, na=False)
              # ID pode ser numérico — converter para string antes do contains
              mask_id = df_lista["ID"].astype(str).str.lower().str.contains(termo, na=False)
              df_lista = df_lista[mask_nome | mask_id]
          ```

       c) Selecionar e ordenar colunas para exibição:
          ```python
          COLUNAS_LISTA = ["ID", "Nome", "Telefone", "GAP", "Faixa_GAP", "Ticket Médio"]
          colunas_existentes = [c for c in COLUNAS_LISTA if c in df_lista.columns]
          df_exibir = (
              df_lista[colunas_existentes]
              .rename(columns={"Faixa_GAP": "Classificação GAP"})
              .sort_values("GAP")
              .reset_index(drop=True)
          )
          ```

       d) Caption com contagem e exibir tabela:
          ```python
          st.caption(f"{len(df_exibir)} clientes encontrados — ordenados por urgência (GAP ascendente). Clique no cabeçalho da coluna para reordenar.")
          if len(df_exibir) > 0:
              st.dataframe(df_exibir, use_container_width=True, hide_index=True)
          else:
              st.info("Nenhum cliente encontrado com os filtros atuais.")
          ```
